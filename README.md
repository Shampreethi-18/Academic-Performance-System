# Evaluation of Academic Performance System

## 📌 Project Overview
This project is designed to evaluate and monitor student academic performance based on CGPA, course completion, and institutional rules.

## 🎯 Objectives
- Calculate Overall CGPA
- Calculate Major CGPA
- Classify Academic Status
- Generate Reports
- Provide Student Dashboard
- Ensure Secure Login & Role-Based Access

## 🧩 Modules Implemented
- Login & Security
- CGPA Calculation (Overall & Major)
- Academic Status Classification
- Notification System
- Report Generation
- Role-Based Access Control

## 🧪 Testing
- 1 Test Plan created
- 1 Test Suite (Functional Testing)
- 19 Test Cases
- Covers both Happy Path & Edge Cases

## ⚙️ Technologies Used
- Java
- Azure DevOps
- Git Repository
- Azure Test Plans

## 📊 DevOps Features
- Source Code Management using Azure Repos
- CI Pipeline using Azure Pipelines
- Test Management using Azure Test Plans
